# GeoExample

## Getting Started

* Get dependencies: `mix deps.get`
* Compile: `mix compile`
* Create Database: `mix ecto.create`
* Run Migrations: `mix ecto.migrate`

## Running

* Start iex: `iex -S mix`
* Add a location: `GeoExample.Query.add_location()`
* List locations: `GeoExample.Query.get_locations()`
